Opdracht 2.3: Onderzoeken

Waarom noem je de eerste HTML-pagina; index.html?

Omdat als een website zonder een specifiek bestand wordt gezocht, dan gaat de browser automatisch zoeken naar de standaardpage/homepage (oftwel index.html). Als je je startpagina niet zo noemt, is er een kans dat de browser niet je homepagina kan vinden. Daarom is het dus belangrijk dat je je startpagina index.html noemt.


Met welke HTML versie werken we nu?

De versie van HTML waar we nu mee werken is 'HTML 5'.


Met welke CSS versie werken we nu?

De versie van CSS waar we nu mee werken is 'CSS 3'.

Welke browser is het populairst en hoeveel procent RAM gebruikt deze browser?

De meest populaire browser is Google Chrome. Als ik Google Chrome open gebruikt het ongeveer 210MB. Dit is ongeveer 2,625% van je RAM (als je 8GB RAM hebt). Ook is het zo als je 10 tabs open hebt staan met een site geladen, ben je ongeveer 900MB van je RAM kwijt. Dit is ongeveer 11,25% van je RAM dat Google Chrome gebruikt.
